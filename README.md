# ゲーム のタイトル：スピードワゴン
## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
主人公のキャラクターがスクロールするステージでジャンプ入力（スペースキー）で障害物を避けたり、アイテムを取ったりします！

## ゲームの実装
###共通基本機能
* 主人公キャラクターに関するクラス
*　背景が流れる

### 担当追加機能
* コインクラス（コイン)
* スコアクラス（コインをとったらスコアが増える）
* 主人公クラス（ジャンプできるように）（基本） 
* 敵クラス
* フィールドクラス（基本）
* スコア数に応じてスキル。（無敵、敵をやっつける、速度上昇など。）
